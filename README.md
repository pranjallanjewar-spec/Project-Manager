# Project Management Web App

This is a full-stack web application that allows users to manage projects efficiently. Users can:

- Create and track project tasks
- Upload completed projects as ZIP files
- Download project documentation in PDF format
- Manage deadlines and task priorities
- Experience a responsive and user-friendly interface

## Technologies Used
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Use
1. Clone or download the repository.
2. Follow backend & frontend setup instructions in your project.
